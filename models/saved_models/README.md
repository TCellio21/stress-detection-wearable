# Saved Models

Trained model checkpoints and serialized models.

## Contents

- Random Forest models (`.joblib`)
- XGBoost models (`.json`, `.bin`)
- Deep learning models (`.pt`, `.h5`)
- Model metadata and training configs

## Naming Convention

`{model_type}_{timestamp}_{accuracy}.{ext}`

Example: `random_forest_20250128_0964.joblib`
